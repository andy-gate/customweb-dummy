# Assets Folder

Folder ini untuk menyimpan file-file assets seperti:

- **Images**: Logo, icons, backgrounds
- **Documents**: PDF templates, document samples
- **Fonts**: Custom fonts (jika ada)
- **Other**: File pendukung lainnya

## Struktur Folder (Rekomendasi)

```
assets/
├── images/
│   ├── logo.png
│   ├── logo-sekneg.png
│   ├── favicon.ico
│   └── backgrounds/
├── documents/
│   ├── templates/
│   └── samples/
├── fonts/
│   └── custom-fonts/
└── README.md (file ini)
```
